# Нейроэволюция для классификации частиц в эксперименте SPD

Нейронная сеть, обученная с помощью генетического алгоритма для бинарной классификации пионов и мюонов в экспериментах физики высоких энергий.

## Обзор проекта

Данное исследование реализует метод эволюционного обучения для классификации частиц в эксперименте SPD (Spin Physics Detector). Модель различает пионы и мюоны на основе 12 трековых параметров с использованием нейроэволюции вместо традиционных градиентных методов.

## Ключевые особенности

- **Эволюционное обучение**: Генетический алгоритм для обучения нейронной сети
- **Специальная функция приспособленности**: Адаптирована для специфики физики частиц
- **Динамический порог**: Автоматическое вычисление порога классификации
- **ReLU активация**: Оптимизированная архитектура для лучшей производительности

## Результаты

- **Пионы**: 95% точности
- **Мюоны**: 89% точности
- **Устойчивость к шуму**: Эффективное разделение реальных частиц и шума
